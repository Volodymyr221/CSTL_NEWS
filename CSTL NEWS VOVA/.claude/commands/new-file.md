# /new-file — Додавання нового JS файлу

Аргумент: назва файлу. Наприклад: `/new-file src/tabs/search.js`

## Крок 1 — Створи файл
Створи файл з базовою структурою в правильному місці в `src/`.

## Крок 2 — sw.js → STATIC_ASSETS (список файлів для кешу)
Відкрий `sw.js`, знайди масив `STATIC_ASSETS`.
Якщо це статичний файл — додай до списку.
Якщо не додати — браузер не кешує файл і PWA (Progressive Web App — веб-додаток що працює офлайн) не працюватиме без інтернету.

## Крок 3 — src/app.js → import (підключення модуля)
Відкрий `src/app.js`, додай імпорт нового файлу.
Порядок імпортів важливий — дотримуйся існуючої структури.

## Крок 4 — index.html (якщо потрібно)
Якщо файл не підключається через bundle — додай `<script>` тег в `index.html`.

## Крок 5 — Оновити CLAUDE.md
Додай новий файл в таблицю "Файлова структура" в `CLAUDE.md` з описом відповідальності.

## Крок 6 — Перевірка
`node --check src/tabs/назва-файлу.js`
