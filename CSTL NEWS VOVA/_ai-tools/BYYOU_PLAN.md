## Знахідки коду (реальні)
- Уся вкладка = `src/tabs/board.js`; обговорення = пости `type:'chat'`, рендер `renderChatCard` (~951-985), dual-root `#disc-content`↔`#board-content` (`getBoardRoot`), стан `discOpen`.
- Картка: 💭 емодзі `board.js:969`; лічильник `👥 N ${partWord}` `:976`; хештеги `.bd-chat-tags` `:952-954`; штрих-лінія = `border-top dashed` на `.bd-chat-foot` (`community.css:2744`); показує ЛИШЕ 1 останнє повідомлення `:957-965`.
- FAB: монолітний у `renderAll` (`:1171-1197`) — ОДИН на обидві вкладки, tab-agnostic; «post»→завжди модалка оголошення; іконка = SVG бульбашки. Хендлери `:1224-1240`.
- Створення обговорення — НЕМАЄ (тип chat прибрано з community-modal 01.07). submitPost(type:'chat') існує в supabase.js, але нічого не викликає.
- «Мої обговорення» — нема. «Збережені» = board-фільтр (saved_posts), кидає на вкладку Дошка, змішано з оголошеннями.
- Дані: пости+коментарі в Supabase (`submitPost`, `fetchMyPosts`, `saved_posts` — уже є, СХЕМУ НЕ чіпаємо).

## Реліз-нотатки (Потік A)

**ЩО ЗМІНИЛОСЬ:**
- Картка обговорення: прибрано 💭 емодзі, прибрано слово «учасник(и)» (лишилось 👥 N), видалено хештеги, показує 2 останніх повідомлення, штрих-лінію зміщено нижче.
- FAB Обговорень ВІДОКРЕМЛЕНО від Дошки (Дошка не змінена). Іконка = червоний круг з білим плюсом. Меню: Створити обговорення / Мої обговорення / Збережені.
- Нова модалка «Створити обговорення» (тема → на модерацію, як оголошення).
- «Мої обговорення» і «Збережені» — окремі кімнати (bottom-sheet) лише для обговорень.

**ЩО МОЖЕ ЗЛАМАТИСЬ:**
- FAB тепер різний на Дошці й Обговореннях — перевірити що на Дошці все як було.
- Нове обговорення = status pending (модерація) → зʼявиться після схвалення (як оголошення).
- Кімнати Мої/Збережені показують лише опубліковані (pending-мої поки не видно — Потік B/окремо).

**ЩО ПЕРЕВІРИТИ НА IPHONE:**
1. Обговорення: картка — без 💭, без хештегів, 👥 без слова, 2 останніх повідомлення, лінія нижче.
2. FAB Обговорень = червоний круг з білим плюсом; меню 3 пункти; на Дошці FAB — як раніше.
3. «Створити обговорення» → модалка → надіслати → тост «надіслано модератору».
4. «Мої обговорення» / «Збережені» → відкриваються кімнати; тап по картці → чат.
5. Чат обговорення відкривається, у шапці нема тегів.
