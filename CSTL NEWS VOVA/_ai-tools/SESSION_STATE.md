## ⚙️ Ключові архітектурні рішення (не змінювати без обговорення)

1. **А+ деплой** — `actions/deploy-pages@v4`, нічого не комітить у `main`
2. **`bundle.js` у git** як робоча база (варіант Б)
3. **`main` як робоча гілка** — пуш = автодеплой
4. **Хук `stop-hook-git-check.sh`** — корисна фіча, не чіпати
5. **Лічильник версії** — `sed` у `deploy.yml`, формат `v{N} · DD.MM HH:MM`
6. **CACHE_NAME у `sw.js`** — оновлювати при КОЖНІЙ зміні статичних файлів. Остання: `cstl-20260417-1145`
7. **Автобуси — зупинки з офіційних квитків VOPAS** — не вигадувати km/ціни без скринів
8. **RSS-парсер** — `scripts/parse_rss.py`, cron щогодини, дедуплікація за URL + заголовком

## 📰 RSS-парсер — поточний стан

### Джерела
| Джерело | Geo | Фільтр |
|---------|-----|--------|
| Суспільне Волинь | Волинь | немає — все |
| Волинь Post | Волинь | немає — все |
| Укрінформ Волинь | Волинь | немає — все |
| Українська правда | Україна | NATIONAL_KEYWORDS |
| Укрінформ Світ | Світ | WORLD_KEYWORDS |
| Українська правда (Світ) | Світ | WORLD_KEYWORDS |

### Логіка
- **Дедуплікація:** за `sourceUrl` + нормалізований заголовок (та сама новина з 2 джерел → одна)
- **Фільтр Україна:** закони/постанови, мобілізація, ЗСУ, тарифи, пенсії, НАТО, санкції тощо
- **Фільтр Світ:** нафта/OPEC, геополітика, війни, G7/ООН, великі катастрофи тощо
- **Олика авто-детекція:** якщо в заголовку/тексті «олика/олицьк» → geo: «Олика»
- **Ліміт:** 100 статей в `data/articles.json`
- **Розклад:** cron `0 * * * *` — щогодини; після нових статей автоматично тригерить `deploy.yml`
