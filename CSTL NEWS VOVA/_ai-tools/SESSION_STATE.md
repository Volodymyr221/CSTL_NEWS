## ⚙️ Ключові архітектурні рішення (не змінювати без обговорення)

1. **А+ деплой** — `actions/deploy-pages@v4`, нічого не комітить у `main` з CI
2. **`bundle.js` у git** як робоча база
3. **`main` як робоча гілка** — пуш = автодеплой
4. **Хук `stop-hook-git-check.sh`** — не чіпати
5. **Лічильник версії** — `sed` у `deploy.yml`, формат `v{N} · DD.MM HH:MM`
6. **CACHE_NAME у `sw.js`** — оновлювати при КОЖНІЙ зміні статичних файлів. **Остання: `cstl-20260419-1630`**
7. **Автобуси — зупинки з офіційних квитків VOPAS** — не вигадувати km/ціни
8. **RSS-парсер** — cron кожні ~30 хв (`0,30 * * * *`), ліміт 15/джерело, MAX_ARTICLES=150, BROWSER_UA + response_headers
9. **Не чіпати зайвий код** — тільки те що явно обговорюється
10. **`.filters-bar` — `position: fixed`** (не sticky!) — через баг iOS Safari з `height:100%` на body

### Cloudflare Worker (проксі між GitHub Actions і сайтом громади)
- **URL Worker:** `https://cstl-proxy.volodymyrshevchuk19.workers.dev`
- **Акаунт Cloudflare:** `volodymyrshevchuk19@gmail.com`
- **Код Worker:** `cloudflare/worker.js` у репо
- **Як працює:** GitHub Actions IP блокується сайтом громади → Worker має інші IP → Worker завантажує HTML і повертає парсеру
- **Функція в парсері:** `parse_gromada_source()` + `gromada_url()` у `scripts/parse_rss.py`
- **Статті через Worker:** `?path=/news/` та `?path=/ogoloshennya-11-12-45-18-02-2021/`

### Логіка парсера
- **Завантаження:** `fetch_rss()` — Chrome BROWSER_UA + `response_headers` → `feedparser.parse(raw, response_headers=...)`
- **Дедуплікація:** за `sourceUrl` + нормалізований заголовок
- **Повний текст:** `content:encoded` → якщо < 600 символів → `fetch_full_article()` зі сторінки
- **Класифікатор:** `classify_entry()` → EVENT_KEYWORDS + майбутня дата → `events.json` або `articles.json`
- **Cron:** `0,30 * * * *` — GitHub безкоштовний тариф може затримувати на 5-30 хв
- **MAX_ARTICLES:** 150, **MAX_PER_SOURCE:** 15, **MAX_EVENTS:** 50
