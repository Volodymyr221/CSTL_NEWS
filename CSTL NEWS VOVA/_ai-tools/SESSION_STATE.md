### Модуль «Відстеження рейсу» (`src/tabs/buses.js`)

1. **`findActiveRoutes()` — пріоритет відстеженого рейсу**
   - Сьогодні: `_trackDate === getTodayISO()` + `ti === -1` → додає рейс з `all` (навіть якщо >90 хв)
   - Майбутні дні: `_trackDate === busDay` → повертає `[tracked]` першим
   - **Причина:** без цього hero показує випадковий рейс замість відстежуваного

2. **`checkTrackNotifications()` — банер для всіх днів**
   - Майбутні дні (`_trackDate > getTodayISO()`): показує «Відстежується» + назва маршруту
   - Сьогодні: повна логіка таймера + сповіщень
   - Минулі дні: кнопка «Відстежити» взагалі не рендериться

3. **Скид снузу при новому відстеженні**
   - `_bannerSnoozedUntil = 0` при кожному натисканні «Відстежити»
   - **Причина:** без цього банер не з'являвся після закриття ✕

4. **`isPastRoute()` — сірі картки в минулих днях**
   - Перша перевірка: `if (busDay < getTodayISO()) return true`
   - **Причина:** без цього вчорашні картки виглядали як сьогоднішні
