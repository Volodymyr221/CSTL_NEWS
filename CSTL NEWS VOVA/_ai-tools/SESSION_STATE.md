## ⚙️ Ключові архітектурні рішення (не змінювати без обговорення)

1. **А+ деплой** — `actions/deploy-pages@v4`, нічого не комітить у `main` з CI
2. **`bundle.js` у git** як робоча база (варіант Б)
3. **`main` як робоча гілка** — пуш = автодеплой
4. **Хук `stop-hook-git-check.sh`** — корисна фіча, не чіпати
5. **Лічильник версії** — `sed` у `deploy.yml`, формат `v{N} · DD.MM HH:MM`
6. **CACHE_NAME у `sw.js`** — оновлювати при КОЖНІЙ зміні статичних файлів. Остання: `cstl-20260417-1500`
7. **Автобуси — зупинки з офіційних квитків VOPAS** — не вигадувати km/ціни без скринів
8. **RSS-парсер** — `scripts/parse_rss.py`, cron щогодини, дедуплікація за URL + заголовком

### Як працює завантаження повної статті
1. Відразу показується excerpt (короткий текст з RSS) + «Завантаження...»
2. Async запит до `allorigins.win/get?url=SOURCE_URL`
3. DOMParser → видалення реклами/скриптів → пошук article-body по сайт-специфічних селекторах
4. Fallback selectors: `[itemprop="articleBody"]`, `article`, `[class*="article-body"]` тощо
5. При успіху — замінює excerpt на повний текст з абзацами, фото, підзаголовками

## 🚌 Модуль «Автобуси» v3 — поточний стан

### Три стани рейсу (завжди по Київському часу)
- `future` — до відправлення: звичайна картка
- `enroute` — в дорозі: зелена рамка, прогрес-бар, позиція
- `past` — прибув: сіра картка

### Часовий пояс
- **Статус рейсу** — розрахунок по `Europe/Kiev` через `Intl.DateTimeFormat`
- **Відображення часу** — конвертується в локальний час користувача через `kyivToLocal()`
- **Закордонні користувачі** — показує «За Києвом HH:MM → HH:MM» дрібним текстом
