## ⚙️ Ключові архітектурні рішення (не змінювати без обговорення)

1. **А+ деплой** — `actions/deploy-pages@v4`, нічого не комітить у `main` з CI
2. **`bundle.js` у git** як робоча база (варіант Б)
3. **`main` як робоча гілка** — пуш = автодеплой
4. **Хук `stop-hook-git-check.sh`** — корисна фіча, не чіпати
5. **Лічильник версії** — `sed` у `deploy.yml`, формат `v{N} · DD.MM HH:MM`
6. **CACHE_NAME у `sw.js`** — оновлювати при КОЖНІЙ зміні статичних файлів. Остання: `cstl-20260418-0302`
7. **Автобуси — зупинки з офіційних квитків VOPAS** — не вигадувати km/ціни без скринів
8. **RSS-парсер** — `scripts/parse_rss.py`, cron кожні 30 хв, дедуплікація за URL + заголовком

## 📰 RSS-парсер — поточний стан

### Джерела
| Джерело | Geo | Фільтр |
|---------|-----|--------|
| Суспільне Волинь | Волинь | немає — все |
| Волинь Post | Волинь | немає — все |
| Конкурент | Волинь | немає — все |
| Укрінформ Волинь | Волинь | немає — все |
| Українська правда | Україна | NATIONAL_KEYWORDS |
| Укрінформ Світ | Світ | WORLD_KEYWORDS |
| Українська правда (Світ) | Світ | WORLD_KEYWORDS |

### Логіка
- **Дедуплікація:** за `sourceUrl` + нормалізований заголовок
- **`strip_html()`** — замінює теги пробілом + `html.unescape()` (декодує `&laquo;` → «)
- **Повний текст:** `content:encoded` з RSS → `get_full_content()` → до 8000 символів
- **Ліміт:** 100 статей в `data/articles.json`
- **Cron:** кожні 30 хв (`*/30 * * * *`)
- **Concurrency:** окрема група `rss-parser` — не конфліктує з `deploy.yml`

## 🚌 Модуль «Автобуси» v3 — поточний стан

### Три стани рейсу (завжди по Київському часу)
- `future` — до відправлення: звичайна картка
- `enroute` — в дорозі: зелена рамка, прогрес-бар, позиція між зупинками
- `past` — прибув: сіра картка

### Часовий пояс
- **Статус рейсу** — розрахунок по `Europe/Kiev` через `Intl.DateTimeFormat`
- **Відображення часу** — конвертується в локальний час користувача через `kyivToLocal()`
- **Закордонні користувачі** — показує «За Києвом HH:MM → HH:MM» дрібним текстом
- **Дні тижня** — `isDayActive(route.days)` фільтрує рейси за `щодня / пн-сб / пн-пт`

### UI
- Поля пошуку «Звідки…» / «Куди…» — без лейблів над ними
