**Концепція:** 3 типи постів в одному блоці
- 🛒 **Дошка** — продам/куплю/шукаю з фото, ціною, категорією (як OLX)

**Спринти:**
- [ ] Спринт 1 (1 тиждень) — Supabase бекенд + адмін-панель /admin
- [ ] Спринт 2 (2 тижні) — фото через Supabase Storage + 3 типи карток + таби-фільтри
- [ ] Спринт 3 (1-2 тижні) — реакції ❤️👍😂😢, збережені, пошук
- [ ] Спринт 4 (опц.) — авторизація (Telegram login / magic-link), профілі, коментарі

**Технічний стек:** Supabase (БД + Storage + Realtime + Auth), без сервера.
