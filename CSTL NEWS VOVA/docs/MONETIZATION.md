## Принцип

**Юзери > монетизація.** Без 500+ щоденних користувачів навіть найкраща
реклама = нуль грн. Зараз фокус на росту аудиторії (Дошка 2.0 + Стратегія
віральності). Монетизація — як наслідок.

**Для громади Олики (~11 800 мешканців, 17 сіл ОТГ)** монетизація реалістична
бо в селі сарафанне радіо швидше за SEO/SMM, конкурси якісної локальної
платформи нуль, лояльність висока.

## 4 джерела доходу (за пріоритетом)

### 1. 🔴 Місцева реклама — найбільше і найшвидше

**Кому продаємо:**
- Магазини Олики (Колос та інші)
- Перукарні, манікюр, СТО
- Кафе, аптеки
- Сезонні (продаж дров, ремонт, такси)
- Локальні фермерські господарства

**Формати:**
- Банер між картками новин у стрічці
- Закріплений «Рекламний пост» на Дошці громади (із бейджем «Реклама»)
- Push-сповіщення про акцію (Фаза 6 — Firebase)
- Згадки у щотижневому дайджесті (Фаза 7 — AI дайджест)

**Ціна реальна для Олики:** 200–800 грн/міс за невеликий бізнес.

**Потенціал:** 10-15 рекламодавців → **3-10 тис грн/міс**.

### 2. 🟡 Підняття оголошень у Дошці громади

Реалізується у Дошці 2.0 (Фаза 9, Спринт 5+).

- Безкоштовне розміщення — як зараз
- Платне підняття: 20-50 грн → вгорі стрічки на 7 днів
- Платне виділення: 30-100 грн → рамка + бейдж «Терміново» / «Топ»
- «Турбо»: 100-200 грн → банер у стрічці + push

**Потенціал:** при 50-100 платних підняттях/міс → **1-5 тис грн/міс**.

### 3. 🟡 Партнерство з адміністрацією

- Олицька сільрада платить за публікацію офіційних оголошень як «офіційний
  канал громади»
- ⚠️ **Конфлікт інтересів:** важко критикувати владу яка платить. Треба
  чіткі правила і прозорість, інакше втратиться довіра аудиторії.
- Реалістична сума: 1-3 тис грн/міс

### 4. 🟢 Гранти і донати

**Гранти:**
- USAID Media Program for Ukraine
- Internews Ukraine
- ZMINA Human Rights Centre
- ZIK Media Hub

Реальна сума: 10-100 тис грн на 6 міс. Велика робота — заявка, звіти.

**Донати:**
- Кнопка «Підтримати» на сайті → Monobank Jar / Privat24
- Реалістично 500-2000 грн/міс при активній аудиторії

## Чого НЕ робити

| Не варто | Чому |
|----------|------|
| ❌ Premium-підписка без реклами | Аудиторія мала. Кожен платний user забере 10 безкоштовних. Бабусі не платять у застосунках |
| ❌ Продаж даних користувачів | Етично спірно для «нашого району». Знищить довіру миттєво |
| ❌ Інвазивна реклама (popup, video autoplay) | Вб'є UX, юзери підуть назад у Viber |
| ❌ Платний доступ до новин | Перший принцип — «замінити хаос Viber безкоштовним інструментом» |

## Що зараз робити для майбутньої монетизації

### Pre-revenue groundwork (зараз, паралельно з Дошкою 2.0)

| ✓ | Задача | Чому важливо |
|---|--------|--------------|
| ☐ | **Підключити аналітику** — Plausible.io (privacy-first, 9$/міс) або безкоштовний Goatcounter | Без цифр (DAU, MAU) не продаси жодній рекламі. Перший день з аналітикою = перший день можливих переговорів |
| ☐ | **Архітектура реклами у Supabase** (Фаза 9 Спринт 1) — таблиця `ads` з полями `id`, `title`, `text`, `image_url`, `link_url`, `placement` (board/news/event), `priority`, `paid_amount`, `expires_at`, `client_email`, `is_active` | Закласти можливість одразу, навіть якщо не використовуємо. Інакше пізніше — рефакторинг таблиць |
| ☐ | **Сторінка `/реклама`** — статична HTML з прайсом + контактна форма «Хочу розмістити» (email на твій ящик) | Точка входу для бізнесу. Без цього бізнеси не знають куди звертатись |
| ☐ | **Кейси у портфоліо** — перші 3-5 безкоштовних реклам своїм знайомим (Smoke_Castle, Kino_Castle, друзі-підприємці) у обмін на дозвіл показувати як кейс | «Ось як виглядає реклама на CSTL NEWS» — продаж легше |
| ☐ | **Прайс-лист на офіційному папері** — простий PDF з форматами і цінами. Роздати власникам бізнесу в Олиці руками | Без прайсу — «а скільки коштує?» → ти не маєш відповіді |

| ✓ | Задача |
|---|--------|

## Архітектура: таблиця `ads` (для Фази 9 Спринту 1)

```sql
CREATE TABLE ads (
  id              BIGSERIAL PRIMARY KEY,
  title           TEXT NOT NULL,
  body            TEXT,
  image_url       TEXT,           -- зі Supabase Storage
  link_url        TEXT,           -- куди веде клік (телефон, сайт)
  placement       TEXT NOT NULL,  -- 'board' | 'news_feed' | 'event_card' | 'banner'
  priority        INT DEFAULT 0,  -- 0 = звичайне, 1+ = підняте
  paid_amount     NUMERIC,        -- скільки заплачено
  client_name     TEXT,
  client_email    TEXT,
  client_phone    TEXT,
  starts_at       TIMESTAMPTZ DEFAULT now(),
  expires_at      TIMESTAMPTZ NOT NULL,
  views_count     INT DEFAULT 0,
  clicks_count    INT DEFAULT 0,
  is_active       BOOLEAN DEFAULT true,
  created_at      TIMESTAMPTZ DEFAULT now()
);

-- Окрема таблиця для аналітики (щоб не блокувати ads оновленнями)
CREATE TABLE ad_events (
  id        BIGSERIAL PRIMARY KEY,
  ad_id     BIGINT REFERENCES ads(id),
  event_type TEXT,                -- 'view' | 'click'
  user_agent TEXT,
  created_at TIMESTAMPTZ DEFAULT now()
);
```

## Ключове питання яке має вирішити Вова

**Чи готовий працювати 4-6 місяців без доходу** поки росте аудиторія?
Спочатку реклама = безкоштовно (кейси). Перші реальні гроші — приблизно
липень-серпень 2026 (після Дошки 2.0 + 500 DAU).

Якщо треба раніше — варіанти:
- Подати на грант (USAID Media Program приймає заявки до X)
- Передплата 1-2 бізнесам авансом за бартер (банер на 6 міс наперед за 3 тис грн)
- Краудфандинг у Viber-чатах громади «підтримай наш проект»

## Що це для бренду

CSTL NEWS не має ставати рекламним порталом — це втратить довіру. Реклама
має бути **обмежена і релевантна** (тільки локальні бізнеси Олики, не
Aliexpress і не казино). 

Принцип: **«ми не продаємо увагу — ми продаємо локальність»**. Це наша
унікальна цінність, яку немає у Google/Facebook.
