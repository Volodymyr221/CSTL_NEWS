Шановні колеги!

Звертається команда проєкту **CSTL LIFE** — безкоштовної інформаційної платформи
(мобільний веб-додаток) для мешканців Олицької громади, смт Олика Волинської області.
Одна з ключових функцій додатка — **актуальний розклад автобусів**, що проходять через
Олику, з можливістю відстеження рейсу та нагадування пасажиру про час відправлення.

Наразі ми відображаємо рейси у напрямку **«до Олики»** (Луцьк → Личани / Олика / Жорнище,
Ківерці → Олика) за відкритими даними. Однак для зручності жителів нам важливо показувати
також **зворотний напрямок — «з Олики»** (у бік Луцька та Ківерців). Ця інформація у
відкритих джерелах представлена неповно, оскільки **Олика є проміжною зупинкою**.

Просимо надати офіційну інформацію по зворотних рейсах, що проходять через Олику:

1. **Час проходження зупинки «Олика»** для кожного зворотного рейсу (Личани → Луцьк,
   Жорнище → Луцьк, Олика → Луцьк, Носовичі → Ківерці) — саме час на зупинці в Олиці,
   а не час відправлення з кінцевої станції.

2. **Останній вечірній рейс** кожного маршруту, що залишається на ніч (напр. у Личанах):
   о котрій годині він вирушає **вранці наступного дня** та коли проходить Олику.

3. **Які рейси повертаються того ж дня**, а які — ні (щоб не показувати неіснуючий рейс).

4. **Дні курсування** кожного зворотного рейсу (щодня / пн–пт / пн–сб / інше).

5. **Перелік і послідовність проміжних зупинок** у зворотному напрямку.

6. **Розкладовий час проходження КОЖНОЇ проміжної зупинки** (не лише Олики, а й
   Дідичів, Носовичів, Метельного тощо — по громаді) для рейсів **в обох напрямках**.
   Зараз ми оцінюємо цей час приблизно (пропорційно відстані), що дає похибку в
   кілька хвилин — офіційний розкладовий час зробив би сповіщення точними.

Ця інформація буде використана виключно для **інформування пасажирів** і сприятиме
зручності користування вашими рейсами. За потреби готові вказати автостанцію як джерело
даних і погодити умови їх використання.
