## Навіщо
1. Реалізувати **приватні групові чати** для локальних спільнот (напр. «Наша Мительне» —
   закритий чат жителів села; ком'юніті Олики). Вступ — за запрошенням + схваленням адміна.
   Контент бачать ЛИШЕ учасники.
2. Винести спілкування у зручне місце таб-бару, не громадячи його (зараз 5 кнопок).

## Контекст (важливі уточнення Вови)
- **Події наразі НЕ для монетизації** — на цьому етапі лише інформування громади про події.
  Монетизація (% з квитків) — пізніше. → не страшно перенести Події у Новини.
- **Віджет подій буде на головній «Громада»** — події отримують видимість і там.

## Фінальна структура навігації (затверджено)

**Таб-бар (5 кнопок):**
```
🚌 Автобуси │ 📋 Дошка │ 🏰 Громада │ 💬 Чати │ 📰 Новини
```
Зміни проти поточного: прибрано окрему вкладку **📅 Події**, додано **💬 Чати**.

- **📋 Дошка** = ЛИШЕ маркетплейс (купити/продати/послуги/віддам). **Обговорення прибрано звідси.**
- **💬 Чати** = хаб спілкування, верхній перемикач 3 підрозділи:
  - **Повідомлення** (приватні 1-на-1; вже існують) — підрозділ за замовчуванням.
  - **Групи** (🆕 приватні групові чати спільнот).
  - **Обговорення** (публічні теми громади; переїхали з Дошки).
- **📰 Новини** = верхній сегмент-перемикач:
  - **Новини** — стрічка статей + гео-фільтр (Всі / Олика / Україна / Світ).
  - **Події** — афіша + фільтр за ЧАСОМ (Майбутні / Минулі).
  - У сегменті «Новини» (категорії **Всі+Олика**) зверху каруселька «🎟 Найближчі події» (свайп).
- **🏰 Громада** = дашборд + віджет найближчих подій + автобуси/світло.

### Рішення-дрібниці (затверджено)
- Назва вкладки спілкування — **«Чати»** (коротке, зрозуміле всім; «Спілкування» довге → обріжеться на малих екранах; «Громада» зайнята).
- Підрозділ Чатів за замовчуванням — **«Повідомлення»**.
- Каруселька подій у Новинах — показувати у **Всі + Олика**.

## Чому саме так (консенсус 4 ШІ)
- **Усі 4** одностайно: Обговорення геть з Дошки → у Чати; Дошка = чистий маркетплейс.
- **Усі 4**: Чати = 3 підрозділи через верхній перемикач, НЕ одна змішана купа.
- **3 з 4**: Чати — окрема вкладка (не в дашборді), бо потрібен бейдж непрочитаних на іконці
  + зрозумілість для старших користувачів.
- Заперечення ШІ «не ховати Події (монетизація)» — **відпало**, бо події зараз інформаційні
  + матимуть віджет на Громаді + каруселька + сегмент = потрійна видимість. Автобуси (щоденна
  утиліта для тих, хто їздить у Луцьк) лишаємо окремою вкладкою.

## ⚠️ Технічні застереження (критично для нашого стеку)
1. **Інвайт-посилання на GitHub Pages (статичний хостинг) → прямий перехід дає 404.**
   Реалізувати **hash-routing** (`https://volodymyr221.github.io/CSTL_NEWS/#/join/<token>`)
   або редирект через `404.html`. Без цього посилання-запрошення ламаються. (Етап 2)
2. **iOS push для груп** — працює лише коли PWA додано на домашній екран (iOS 16.4+, APNs через
   Safari). У нас push для автобусів/чату вже працює — частково покрито; потрібен онбординг.
3. **Realtime-підписки (WebSocket) закривати при зміні вкладки** — інакше memory leak (витік
   пам'яті) на ванільному JS. Дисципліна вже є (аудит чату 24.06) — дотримати для груп.

## Ризики для Етапу 2 (груп) — врахувати при дизайні
- **«Мертві групи»** (мала село-група замовкає) → бот-дайджест (погода/світло) для активності.
- **Порожній екран груп у новачка** → картка «Тут будуть чати твого села — попроси сусіда
  додати / як це працює».
- **Модерація приватних груп** → кнопка «Поскаржитись адміну Olyka» навіть усередині приватного
  чату (це бренд Olyka Castle).
- **Приватність** → бейдж «Приватна» всюди; групи НЕ в публічному пошуку/списку.
