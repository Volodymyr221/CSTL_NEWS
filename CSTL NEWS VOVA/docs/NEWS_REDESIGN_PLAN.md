# NEWS_REDESIGN_PLAN — перебудова розділу Новини (рішення Роми 05.07.2026)

> Велика перебудова: AI-агент новин + перенос UI у Громаду + нова вкладка «Шо там».
> Розбито на 3 блоки, кожен деплоїться окремо. База знань громади → `HROMADA_KB.md`.

## Візія (навіщо)
Google News викидаємо (ховає справжні URL, крихко). Натомість — **AI-агент**, що
розумно шукає, фільтрує й дедуплікує новини зі справжніми посиланнями. Новини
переїжджають усередину вкладки Громада окремим блоком. На звільнене місце —
вкладка «Шо там» (контент Алли + свята з AI-статтею й картинкою).

---

## БЛОК A — AI-агент новин (бекенд, зона Роми)
Новий робітник GitHub, що замінює Google News. Три «місії» за запуск, справжні URL.
1. Конфіг громади → `scripts/hromada_config.json` (з `HROMADA_KB.md`: 17 НП, ключі, джерела).
2. `scripts/ai_news_agent.py` — каркас: Anthropic API + інструмент web_search → JSON новин {title,url,geo,date,summary,why}.
3. **Місія Громада** (перейм. з «Олика»): баланс 50% Олика / 50% села; теми — люди/пам'ятки/населення/культура/події; дедуп за змістом проти наявних.
4. **Місія Волинь**: 4-5 джерел; **семантичний дедуп** (AI бачить що новина та сама, навіть іншими словами).
5. **Місія Україна+Світ**: 5-6 джерел, 50/50; лише серйозне (геополітика/війна/важливе), без «папуг у Таїланді».
6. 🔴 Анти-SSRF: домен-whitelist → перевірка «URL публічний, не приватний IP» — щоб будь-яке справжнє видання проходило безпечно.
7. Повний текст із реального URL (`fetch_full_article` + фікс bs4 4.15 `attrs=None`) + денні ліміти/мердж у `articles.json`.
8. 🔴 Перейменувати geo «Олика»→«Громада» (міграція даних + `section_of` + фільтри).
9. Workflow `ai-news-agent.yml` (cron ~3 год) + secret `ANTHROPIC_API_KEY`.

## БЛОК B — Новини всередину вкладки Громада (зона Роми, UI)
10. Винести стрічку новин у НОВИЙ блок унизу вкладки Громада (розмір як «Корисні контакти»).
11. Згори блока — 3 кнопки-фільтри: **Громада · Волинь · Україна та світ**.
12. 🔴 Вкладена НЕЗАЛЕЖНА прокрутка карток усередині блока (nested scroll) — без конфлікту зі скролом сторінки Громада (делікатно на iOS).
13. Прибрати «Новини» з нижнього таб-бару (5 вкладок лишається — місце під «Шо там»).

## БЛОК C — Вкладка «Шо там» (на місці Новин)
14. Нова вкладка: стрічка контенту Алли (редактор) + свята.
15. Свята: AI-стаття про свято (історія / притча / традиція).
16. 🔴 AI-згенерована картинка до свята — **Anthropic картинки НЕ генерує**; треба окремий image-API (напр. OpenAI/Stability) або банк готових зображень. Рішення відкрите.
