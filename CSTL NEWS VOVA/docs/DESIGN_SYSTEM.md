## ⚠️ Помилки які були зроблені — читати перед будь-якою зміною стилів

### Проблема: padding на дочірніх елементах не звужував вміст

**Що сталось:** Вова просив додати горизонтальні відступи (відстань від краю) всередині модалок. Було зроблено кілька спроб — жодна не давала видимого результату.

**Що реально вирішує проблему:**
```css
/* НА OUTER PANEL (той що має overflow:hidden і border-radius): */
padding: 0 20px;

/* scroll container (контейнер прокрутки) — тільки вертикальний padding: */
padding: 28px 0;
```

**Правило:** padding (внутрішній відступ) завжди ставити на **outer panel** (зовнішня панель — та що має `overflow:hidden`), а не на scroll-контейнері всередині.

## Кольорова палітра (набір кольорів)

```css
:root {
  --red:        #C41E3A;   /* основний акцент */
  --red-dark:   #9e1830;   /* active/hover стан кнопок */
  --black:      #1a1a1a;   /* основний текст */
  --gray:       #666666;   /* другорядний текст, дати, підписи */
  --gray-light: #f5f5f5;   /* фони, кнопки закриття */
  --border:     #e5e5e5;   /* межі карток, лінії */
  --white:      #ffffff;   /* фон сторінки */
}
```

| Роль | CSS-змінна | HEX |
|------|-----------|-----|
| Основний акцент | `var(--red)` | `#C41E3A` |
| Hover акценту | `var(--red-dark)` | `#9e1830` |
| Основний текст | `var(--black)` | `#1a1a1a` |
| Другорядний текст | `var(--gray)` | `#666666` |
| Фон чіпів | `var(--gray-light)` | `#f5f5f5` |
| Межі | `var(--border)` | `#e5e5e5` |
| Фон сторінки | `var(--white)` | `#ffffff` |

## Візуальний стиль

- **Мінімалізм:** білий фон, тонкі межі, без тіней на картках
- **Картки:** `border-radius: 12px`, `border: 1px solid var(--border)`
- **Модальне вікно:** `border-radius: 20px 20px 0 0` (rounded top corners)
- **Акцентний колір:** тільки на важливих елементах (активний фільтр, дата, ексклюзив)
- **Ексклюзив:** червона межа картки + бейдж "ЕКСКЛЮЗИВ" на червоному фоні

## Компоненти

### Шапка (Header)

```
┌─────────────────────────────────┐
│ CSTL NEWS              ☁️ 5°   │  ← height: 56px (var(--header-h))
└─────────────────────────────────┘
```

- `position: fixed; top: 0` — завжди вгорі
- Логотип: "CSTL " чорний + "NEWS" червоний, Georgia serif 20px
- Погода: emoji + температура, fetch з open-meteo.com при завантаженні

---

### Гео-фільтри (Chips)

```
[Всі●] [Олика] [Волинь] [Україна] [Світ]
```

- Горизонтальний скрол, `scrollbar-width: none`
- Активний чіп: `background: var(--red)`, `color: white`
- Неактивний: `border: 1px solid var(--border)`, `color: var(--black)`
- `border-radius: 20px`, `padding: 5px 12px`, `font-size: 13px`

### Toast (маленьке сповіщення)

```
         ╔══════════════════╗
         ║  Новину надіслано ║  ← над tab bar
         ╚══════════════════╝
```

- `position: fixed; bottom: calc(64px + 12px)`
- `border-radius: 20px`, чорний фон, білий текст
- Показується 3 секунди, потім ховається (`opacity: 0`)

### Splash Screen (заставка)

```
    ╔═══════════╗   ← червоний фон
    ║ [замок]   ║   ← logo.png 230×230px, анімація shadowSweep
    ║ CSTL NEWS ║   ← Georgia serif 42px, анімація textShadowSweep
    ║ Олика·... ║   ← 14px, opacity 0.7
    ╚═══════════╝
```

- Тривалість: 3.5s → `opacity: 0` → `remove()` після 0.6s fade-out
- `padding-bottom: 150px` — контент підвищено відносно центру
- Анімації: `splashPop` (поява лого), `splashFadeUp` (поява тексту), `shadowSweep` (тінь лого), `textShadowSweep` (тінь тексту)
