# /seo-audit — SEO перевірка CSTL NEWS

Ти — SEO-аудитор сайту **CSTL NEWS** (https://volodymyr221.github.io/CSTL_NEWS/).

## Що перевіряємо

### 1. Технічний SEO (перевірити в коді)

**Мета-теги (index.html):**
- [ ] `<title>` — є? Унікальний? До 60 символів?
- [ ] `<meta name="description">` — є? До 160 символів? Містить ключові слова?
- [ ] `<meta name="keywords">` — є? (необов'язково, але бажано)
- [ ] Open Graph теги: `og:title`, `og:description`, `og:image`, `og:url`
- [ ] Twitter Card теги

**Структура:**
- [ ] Є тільки один `<h1>` на сторінці?
- [ ] Ієрархія заголовків h1 → h2 → h3 правильна?
- [ ] Всі зображення мають атрибут `alt`?
- [ ] Є `robots.txt`?
- [ ] Є `sitemap.xml`?

**PWA і швидкість:**
- [ ] `manifest.json` заповнений (name, description, icons)?
- [ ] Service Worker кешує правильно?
- [ ] Зображення оптимізовані (WebP)?

**Посилання:**
- [ ] Немає битих посилань (404)?
- [ ] Канонічний URL встановлений (`<link rel="canonical">`)?

### 2. Контентний SEO

**Заголовки статей:**
- Чи містять ключові слова?
- Формат: "Що [подія] — [місце/хто] [деталь]"
- Приклад: "Ремонт доріг в Олиці 2024: що відомо"

**Ключові слова для CSTL NEWS:**
Основні: "Олика", "Олика новини", "CSTL News", "Волинська область новини"
Додаткові: "замок Олика", "Олика сьогодні", "Волинь події"
Локальні: "[назва вулиці] Олика", "[назва події] Олика"

**Мета-опис статті:**
Формат: "[Що сталось] в [де] [коли]. [Деталь що зацікавить читача]."
Довжина: 120-160 символів

### 3. AI SEO (оптимізація під AI-пошук)

Пошуковики з AI (Google AI Overview, ChatGPT, Perplexity) цитують сторінки де є:
- Чіткі визначення (FAQ формат)
- Структуровані дані (списки, таблиці)
- Відповіді на конкретні питання

**Додати до головної сторінки:**
```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "NewsMediaOrganization",
  "name": "CSTL NEWS",
  "url": "https://volodymyr221.github.io/CSTL_NEWS/",
  "description": "Локальні новини Олики та Волинської області",
  "areaServed": "Олика, Волинська область, Україна"
}
</script>
```

**Для кожної статті додати:**
```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "NewsArticle",
  "headline": "[ЗАГОЛОВОК]",
  "datePublished": "[ДАТА]",
  "author": {"@type": "Organization", "name": "CSTL NEWS"},
  "publisher": {"@type": "Organization", "name": "CSTL NEWS"}
}
</script>
```

### 4. Алгоритм аудиту

Коли викликаний `/seo-audit`:

1. Прочитай `index.html` — перевір мета-теги
2. Прочитай `manifest.json` — перевір PWA-мета
3. Перевір наявність `robots.txt` і `sitemap.xml`
4. Перевір структуру заголовків h1-h3
5. Перевір alt-теги у зображень
6. Видай список: ✅ ОК / ⚠️ Потребує уваги / ❌ Критично
