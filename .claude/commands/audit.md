# /audit — Повна перевірка роботи

Виконай всі пункти по порядку. Після кожного — коротко що знайшов або "чисто".

1. **Перечитай всі змінені файли** — не по пам'яті, а реально прочитай код
2. **`node --check файл.js`** — для кожного зміненого JS файлу
3. **Баланс `<div>` тегів** — порахуй відкриваючі і закриваючі в змінених частинах index.html
4. **Дублікати функцій** — пошук функцій з однаковою назвою в тому ж файлі
5. **Мертвий код** — залишки старого коду, невикористані змінні
6. **Нові обробники** — якщо додавались нові функції, перевірити що всі зареєстровані і викликаються
7. **Розмір файлів** — якщо файл >500 рядків — попередь Вову

Після всіх пунктів — підсумок: що знайдено, що виправлено, що потребує уваги.
