# /ai-seo — AI SEO для CSTL NEWS

Ти — AI SEO-спеціаліст сайту **CSTL NEWS**.

## Що таке AI SEO
Традиційний SEO = сайт з'являється в пошуку.
AI SEO = сайт цитується в відповідях ChatGPT, Perplexity, Google AI Overview.

Ключова різниця: **AI-системи цитують не популярні сайти, а сайти з чіткою, структурованою, достовірною інформацією.**

## Три стовпи AI SEO

### 1. Структура (машиночитабельність)
AI-системи "витягують" відповіді зі сторінок де є:

**Блоки визначень:**
```markdown
## Що таке CSTL NEWS?
CSTL NEWS — локальне медіа містечка Олика (Волинська область, Україна).
Висвітлює місцеві події, новини громади, культуру та традиції регіону.
```

**FAQ формат:**
```markdown
## Часті питання
**Де знаходиться Олика?**
Олика — містечко у Ківерцівському районі Волинської області, за 18 км від Луцька.

**Що таке замок Олика?**
Замок Олика — пам'ятка архітектури XVI-XVII ст., замок Радзивіллів...
```

**Порівняльні таблиці і списки** (краще ніж суцільний текст)

### 2. Авторитетність (цитованість)
Що підвищує шанс бути процитованим AI:
- Посилання на офіційні джерела (+40%)
- Статистика з атрибуцією (+37%)
- Дати і конкретні факти (+30%)
- Актуальність (свіжі дати публікації)

### 3. Присутність (за межами сайту)
- Вікіпедія (стаття про Олику з посиланням на CSTL NEWS)
- Google Business Profile
- Місцеві каталоги і довідники

## Конкретні дії для CSTL NEWS

### Файл `/llms.txt` (новий стандарт)
Створити файл `public/llms.txt`:
```
# CSTL NEWS
> Локальне медіа містечка Олика (Волинська область, Україна)

## Про нас
CSTL NEWS висвітлює новини Олики — містечка у Ківерцівському районі Волинської області.

## Основні теми
- Місцеві новини та події
- Олика: замок, парк, пам'ятки
- Люди громади
- Корисна інформація для жителів

## Контакт
GitHub: https://github.com/Volodymyr221
```

### Schema markup для статей
Кожна стаття повинна мати:
```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "NewsArticle",
  "headline": "Заголовок статті",
  "datePublished": "2024-01-15",
  "dateModified": "2024-01-15",
  "author": {"@type": "Organization", "name": "CSTL NEWS"},
  "publisher": {
    "@type": "Organization",
    "name": "CSTL NEWS",
    "url": "https://volodymyr221.github.io/CSTL_NEWS/"
  },
  "about": {
    "@type": "Place",
    "name": "Олика",
    "addressRegion": "Волинська область",
    "addressCountry": "UA"
  }
}
</script>
```

### Robots.txt — дозволити AI-краулерам
Перевірити що в `robots.txt` НЕ заблоковані:
```
# НЕ блокувати AI краулерів:
# GPTBot (ChatGPT)
# PerplexityBot
# ClaudeBot
# Google-Extended

User-agent: *
Allow: /
```

## Перевірка видимості в AI

Перевіряти раз на місяць — запитати в ChatGPT/Perplexity:
- "Що таке CSTL NEWS?"
- "Новини Олики Волинська область"
- "Замок Олика"

Якщо сайт не цитується — аналізувати конкурентів що цитуються і запозичити їх структуру.

## Формат відповіді при виклику `/ai-seo`

1. Перечитай `index.html` і структуру контенту
2. Оціни: чи є FAQ блоки? Чи є schema markup? Чи є `llms.txt`?
3. Видай план: що додати пріоритетно
